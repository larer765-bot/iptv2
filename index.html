Guide : Pourquoi les chaînes bloquent ?

Les navigateurs web modernes ont des règles de sécurité très strictes appelées CORS. Si l'hébergeur de la chaîne de télévision n'a pas autorisé les sites tiers, le navigateur refuse de lire le flux.

La solution magique (Extension de navigateur)

Pour contourner cette sécurité et faire fonctionner 100% des flux, il vous suffit d'installer une extension d'un clic pour désactiver temporairement CORS sur votre navigateur.

Comment faire ?

1
Installez l'extension CORS :
Télécharger pour Chrome, Edge, Brave Télécharger pour Firefox
2
Activez l'extension : Cliquez sur l'icône de l'extension installée dans votre barre d'outils et assurez-vous de l'allumer (elle devient généralement verte).
3
Actualisez la page : Rechargez notre site et lancez votre chaîne. Elle démarrera instantanément !

Autre cause : Contenu mixte (HTTP sur HTTPS)

Si l'adresse de notre application commence par https://, le navigateur bloquera les flux commençant par http://. Pour éviter cela, lancez notre application localement sur votre ordinateur ou téléchargez l'index.html pour l'exécuter en double-cliquant dessus.
